> In this section, you will learn how to visualise the outputs of the
> previous ('Model prediction') section which produces outputs that are
> mult-band rasters.

Each band of these multi-band rasters are a single class. So, each class can be displayed individually as binary maps (present/absent).

Visualise multi-band images as single bands in QGIS:

<br>

**Step 1:** Open all the tiles in the outputs in QGIS.

<br>

**Step 2:** 
```
Select one tile > change from Multiband Color to Paletted/Unique values
```
![enter image description here][1]

<br>

**Step 3:** Select a band corresponding to the land use class which needs to be looked at:

![enter image description here][2]

<br>

**Step 4:** Select a colour scheme in the color ramp:

![enter image description here][3]

<br>

**Step 5:** Press Classify and you get this:

![enter image description here][4]

<br>

**Step 6:** Press `Apply` on the tab from the previous step.

<br>

**Step 7:**
```
Right click on that layer > Styles > Copy styles
```
![enter image description here][5]

<br>

**Step 8:** 
Press shift and select the first and last tile (to select all the tiles at once) and paste style to the final map.

![enter image description here][6]

<br>

**Step 9:** 

It will look like the following (ignore the missing chunk. This is from one of the earliest iterations). This is a binary map (presence = yellow; absence = dark blue) of dense forests. 

![enter image description here][7]

```
Raster > Miscellaneous > Merge
```
